# **n8n**
n8n is a no-code or low code automation tool that let's you connect apps, APIs and data without complex code implementation.
It helps you automate tasks, save time and reduce manual errors in your daily workflows.
Create connections between your favorite tools and build automated workflows visually with a drag and drop interface.


## **How n8n stands out?**
1. Self hostable (run on your server) or you can also use the cloud interface
2. 300+ pre built integrations tools like GoogleSheet, Airtable etc
3. Built in support for custom API call to connect to almost any app
4. Scalable , from small workflow to complex automation systems
5. Advanced ai integration for intelligent workflow automation

## **How it works?**
1. Trigger starts when a new data enters
2. Data flows through processing nodes
3. Business logic transform information
4. Output deliverd to definition

**THE WORKFLOW CAN BE ADOPTED TO MULTIPLE USE CASES BY CHANGING THE DATA SOURCE AND DESTINATION NODES**


## **Installation**
There are two ways to install or use the n8n.
First is cloud interface which is on browser, that's a paid service after 14 days trial.
Second one is completely free which is on localhost, but there are some limitation , as we are not able to share our localhost work
to everyone.
If you are someone who wants to learn n8n use second way.
For that you need to install node in your device first.

Download link for node : ```https://nodejs.org/en/download```
Install it in your device.


After installing open cmd or command prompt and write following code:-
```npm install g- n8n```

Now, wait for 5-10 mins, as it will download many packages.

Now we need to login in or sign up for n8n, as soon the downloading finish.
Write code :- 
```n8n start```

When it run first time it will create or sign in your n8n accont and provide you a activation key on gmail. Use that key and paste in the login page.
Your localhost n8n is setup.

To run the n8n, locally always open cmd and write 'n8n start' to start the application locally.
