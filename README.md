# **n8n**
n8n is a no-code or low code automation tool that let's you connect apps, APIs and data without complex code implementation.
It helps you automate tasks, save time and reduce manual errors in your daily workflows.
Create connections between your favorite tools and build automated workflows visually with a drag and drop interface.


## **How n8n stands out?**
1. Self hostable (run on your server) or you can also use the cloud interface
2. 300+ pre built integrations tools like GoogleSheet, Airtable etc
3. Built in support for custom API call to connect to almost any app
4. Scalable , from small workflow to complex automation systems
5. Advanced ai integration for intelligent workflow automation

## **How it works?**
1. Trigger starts when a new data enters
2. Data flows through processing nodes
3. Business logic transform information
4. Output deliverd to definition

**THE WORKFLOW CAN BE ADOPTED TO MULTIPLE USE CASES BY CHANGING THE DATA SOURCE AND DESTINATION NODES**
